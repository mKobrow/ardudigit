						int buttonPin = 7;
						int ledPin = 6;

						void setup() {
						  pinMode(buttonPin, INPUT_PULLUP);
						  pinMode(ledPin, OUTPUT);
						}

						void loop() {
						  if (digitalRead(buttonPin) == LOW) {
						    digitalWrite(ledPin, HIGH);
						    delay(5000);
						    digitalWrite(ledPin, LOW);
						  }
						}

Das ist der gesamte Code. Die einzelnen Zeilen werden auf den nächsten Folien erläutert.

Die if-Verzweigung überprüft, ob der Wert auf dem Button-Pin LOW ist, also ob der Button gedrückt ist.


						int buttonPin = 7;
						int ledPin = 6;

						void setup() {
						  pinMode(buttonPin, INPUT_PULLUP);
						  pinMode(ledPin, OUTPUT);
						}

						void loop() {
						  if (digitalRead(buttonPin) == LOW) {
						    digitalWrite(ledPin, HIGH);
						    delay(5000);
						    digitalWrite(ledPin, LOW);
						  }
						}

Falls das so ist, wird der LED-Pin auf HIGH gesetzt, also die LED angeschaltet. Dann wird 5 Sekunden gewartet und die LED wird anschließend wieder ausgeschaltet.
